Usage

  1. Download the css file from the Github Repo and include it in your page. 
    <link rel="stylesheet" href="hint.css">
  2. Any element on which you have to show the tooltip needs to be given a tt class and the tooltip text should be written in the data-title attribute. 
    This will show the default <span class="tt" data-title="Hello!">tooltip</span>
